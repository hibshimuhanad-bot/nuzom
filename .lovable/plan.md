

# إثراء صفحة "من نحن" بمحتوى أكثر

## المشكلة
الصفحة حالياً تحتوي فقط على: قصة قصيرة، رؤية ورسالة، و3 بطاقات تقنية — محتوى قليل لا يعطي صورة كاملة عن الشركة.

## الإضافات المقترحة

### 1. توسيع قسم القصة
- تحويل النص القصير إلى فقرتين: تأسيس الشركة + التركيز على السوق السعودي والقطاعات المستهدفة (مقاولات، قانون، إدارة مرافق)

### 2. إضافة قسم "قيمنا" (Our Values)
4 بطاقات بأيقونات:
- **البساطة**: أنظمة جاهزة بدون تعقيد
- **الأمان**: حماية بيانات العملاء أولوية
- **الابتكار**: استخدام أحدث التقنيات والذكاء الاصطناعي
- **الدعم المستمر**: فريق معك من التفعيل للتشغيل

### 3. إضافة قسم "أرقامنا" (By the Numbers)
شريط إحصائيات بصري:
- 4 أنظمة جاهزة
- +5 قطاعات نخدمها
- تفعيل خلال أيام
- دعم فني 24/7

### 4. إضافة قسم "أنظمتنا" (Our Products)
عرض سريع للمنتجات الأربعة (HSSE, Legal, Tasks, Booking) مع أيقوناتها وأسمائها + رابط لكل منتج — مأخوذ من `products.ts`

### 5. إضافة CTA في الأسفل
زر "تواصل معنا" يوجه لصفحة التواصل

### التغييرات التقنية
- **`src/pages/About.tsx`**: إضافة الأقسام الأربعة الجديدة + توسيع القصة
- **`src/contexts/LanguageContext.tsx`**: إضافة مفاتيح ترجمة جديدة للقيم والأرقام
